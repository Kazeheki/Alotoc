I put my clock somewhere I cannot access it anymore. Reopening is not helping.

Reopening won't work since the clock saves it position on closing. But you can delete the storage file so that the default values will be loaded. To do so follow these steps:

Windows
  1. Press Windows-Key + r
  2. Type %appdata% and press enter
  3. Go up one folder (into AppData)
  4. Open Local
  5. Delete folder .Alotoc
