Ever wanted a cool clock that will never vanish behind stupid windows and frames?

Now you can get it!

Alotoc (short for: Always on top clock) is the clock you always wanted! It will always stay on top of everything.

Useful for:

  • Gamers playing in fullscreen mode and being annoyed by the "Windows" clock only being on the main screen
  • Anyone else, annoyed by the "Windows" clock only being on the main screen ;P
  • Anyone who wants a bigger clock, anywhere on the screen


Features

  • ALWAYS on top of everything (except you or Windows minimize it)
  • Draggable
  • Color is selectable (you can type in the hex-value of a color if you want to)
  • Displays the same time as your standard computer clock



How to use?

You can download the current version and source code here. But you only need the Alotoc.jar file which you can download directly here.

Now you simply have to execute the .jar file. Once done the clock apears right where your cursor is.

At the top right corner you can find a "X" marking the quit button and an "O" marking the options/preferences. The preferences button will open a new window where you can select a preset color or type a hex-value of any color you want. If you typed in a color, don't forget to hit the "GO" button! Both the clock and the preferences window are draggable to wherever you want them to be.
